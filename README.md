# granite-embeddings
A simple Gradio app that hosts endpoints on HuggingFace in the following space: https://aploetz-granite-embeddings.hf.space/embed. Can be run locally, and accessed at http://localhost:7860 .

## execution
```bash
python3 app.py
```
